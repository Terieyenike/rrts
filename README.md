# React Redux with TypeScript

An app that displays all todos when the fetch button is clicked and you can remove a todo from the list when clicked upon.

## Running the project
`yarn start`

## Dependencies
- Axios
- Redux
- React-Redux
- Redux-thunk

### devDependencies
- @types/react
- @types/react-dom
- typescript
